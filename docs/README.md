# Documentation

Check out the airflow-etl-playbook site again at the same level as airflow-etl-playbook:

`git clone git@github.com:Pravalika20007/airflow-etl-playbook.git airflow-etl-playbook-docs`

check out the gh-pages branch there:

`git checkout gh-pages`


Rebuild with sphinx from airflow-etl-playbook repo:

`sudo apt-get install python3-sphinx`

`pip install sphinx_rtd_theme`

`make html`

( or `./rebuild_all.sh` )
